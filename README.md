# Web Scraping Portfolio

Python web scraping projects built using BeautifulSoup, Selenium, requests, and pandas.

## Projects

### 1. Books Scraper — BeautifulSoup
- Scraped 1000 books from books.toscrape.com
- Extracted: Title, Price
- Pages scraped: 50 pages automatically
- Tools: requests, BeautifulSoup, pandas

### 2. Quotes Scraper — BeautifulSoup
- Scraped 100 quotes from quotes.toscrape.com
- Extracted: Quote, Author, Tags, Page Number
- Pages scraped: 10 pages
- Tools: requests, BeautifulSoup, pandas

### 3. Laptop Scraper — Selenium
- Scraped laptop data from webscraper.io
- Extracted: Name, Price, Rating, Reviews
- Tools: Selenium, pandas

### 4. Login Automation — Selenium
- Automated login flow using Selenium
- Tools: Selenium, WebDriverWait
